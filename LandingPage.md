# 🌐 Gali'Pet Landing Page

## 📋 Summary
Static marketing website showcasing Gali'Pet platform features, benefits, and call-to-action for users to download the mobile app or access the dashboard.

---

## 🛠️ Tech Stack

| Technology | Version | Purpose |
|---|---|---|
| **React** | 18.x | UI library for interactive components |
| **Vite** | 5.x | Fast build tool & dev server |
| **Tailwind CSS** | 3.x | Utility-first CSS framework for styling |
| **React Router** | 6.x | Client-side routing between pages |
| **Framer Motion** | 10.x | Smooth animations & transitions |
| **TypeScript** | 5.x | Type safety for JavaScript |

---

## 🎯 Why These Dependencies?

- **React**: Industry standard for building interactive UIs
- **Vite**: 10x faster than Webpack, instant HMR (Hot Module Replacement)
- **Tailwind CSS**: Rapid UI development with pre-built utility classes
- **React Router**: Seamless page navigation without full page reloads
- **Framer Motion**: Professional animations to engage visitors
- **TypeScript**: Catch errors early, better code maintainability

---

## 📱 How It Works (Step-by-Step)

1. **User visits landing page** → Vite serves optimized static assets
2. **Page loads** → React renders components with Tailwind styling
3. **User scrolls** → Framer Motion triggers smooth animations
4. **User clicks CTA** → React Router navigates to app download or dashboard login
5. **Page optimized** → Vite bundles only used code (tree-shaking)

```bash

npm run build    # Production build
npm run preview  # Preview production build
```
